# SmartRegener – Regenerative DC Motor Controller

SmartRegener is a regenerative braking capable brushed DC motor controller developed as part of the TÜBİTAK 2209-A research program.

## Features

- STM32G474 microcontroller
- DRV8701 H-Bridge gate driver
- Regenerative braking capability
- Current sensing with INA240
- Reverse polarity protection
- Overcurrent protection
- ESP32 communication interface
- 6-layer PCB architecture

## Hardware Specifications

| Parameter | Value |
|------------|------------|
| Input Voltage | 12 V |
| Motor Power | 120 W |
| MCU | STM32G474 |
| Communication | ESP32 |
| PCB | 6 Layer |
